# Maintenance Tracker

A personal iPhone-friendly maintenance task tracker built with React and Vite.

## Features

- Add, edit, delete, and complete maintenance tasks
- Store tasks locally in the browser
- Search and filter tasks
- Export tasks to Excel-friendly CSV
- Import edited CSV files back into the app
- Mobile-first layout for iPhone

## Deploy to Vercel

1. Create a new GitHub repository.
2. Upload all files from this project.
3. Go to Vercel and import the GitHub repository.
4. Use the default Vite settings:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
5. Deploy.

## Run locally

```bash
npm install
npm run dev
```
